background-repeat

배경으로 들어간 이미지는 스크린리더기에 읽히지않는다. 기본적으로 배경이미지는 왼쪽 위부터 반복된다. 하나만 적용하기 위해서는 반복을 끄는 background-repeat: no-repeat;을 쓴다.

background-position

왼쪽 정렬 표시방법

background-position: 0 50%;
= background-position: left center;
