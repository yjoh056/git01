html>head>body

작성시 같은 태그끼리 같은 선에 들어가도록 정리

head : 웹페이지에 정보 전달 역할

head요소에 포함할 수 있는 요소들

meta : HTML 문서의 설명, 키워드 문서의 작성자 지정

title : HTML 문서의 제목 지정 (검색엔진에 검색되는 첫 요소)

link : 외부 CSS파일을 연결할때 사용

style : 내부 CSS를 정의할 때 사용
