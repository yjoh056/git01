position

Float 속성
값 static | reative | absoulute | fixed | sticky
(기본 | 상대좌표 | 절대좌표 | 고정좌표 | 스티키)

사용가능 속성

1. static : 기본

2. relative : 상대좌표 → 자기 자신의 좌측상단

3. absolute : 절대좌표 → 브라우저 좌측상단

4. sticky : [스크롤시 고정](https://github.com/w3c/csswg-drafts/issues/865)

5. fixed : 고정

위치값 지정

1. top, left, right, bottom

2. z-index 로 깊이지정 (z-index 는 값이 클수록 먼저보임)
