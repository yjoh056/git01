flex

flex container(부모속성) 주축의 방향은 flex container의 flex-direction 속성으로 결정
flex는 반드시 부모 container에 적용하기!
값 flex-direction | flex-wrap | justify-content | align-items | align-content
flex item(자식속성) flex item은 주축(main axis)에 따라 정렬.
flex-direction 기본값 : row
row는 주축의 방향을 왼쪽에서 오른쪽 방향으로 흐르게 한다.
column은 주축의 방향을 위에서 아래 방향으로 흐르게 한다.
값 flex | flex-grow | flex-shrink | flex-basis | order

flex-direction 방향

{flex-direction: row;} /* 좌->우 */
{flex-direction: row-reverse;} /* 우->좌 */
{flex-direction: column;} /* 상 -> 하 */
{flex-direction: column-reverse;} /* 하 -> 상 */
